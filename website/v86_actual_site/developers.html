Mission contract anatomy

A prompt is not a contract.

A contract defines objective, scope, owner, risk, evidence, done criteria, and promotion rules.

{ "mission": "produce-reviewable-proof", "objective": "turn an AI-agent task into an Evidence Docket", "doneCriteria": ["claims bounded", "sources attached", "risk ledger complete", "replay path present"], "promotionRule": "accepted proof only" }
