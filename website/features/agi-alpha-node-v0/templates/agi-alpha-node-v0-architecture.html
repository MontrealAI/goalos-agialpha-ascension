TEN CONSTITUTIONAL PLANES

Power is decomposed before it is composed.

No single role may silently define identity, mission, resources, route, evidence, verification, and authority. Each plane emits an artifact and hands control to the next bounded role.

STATE MACHINE / TEN PROOF GATES

The node advances only through inspectable gates.

Every stage has a named state, accountable agent, machine-readable artifact, plain-language explanation, and explicit stop condition.
