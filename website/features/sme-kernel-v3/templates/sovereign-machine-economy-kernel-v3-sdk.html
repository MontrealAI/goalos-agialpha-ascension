Adapter contract

Use any intelligence source. Keep one constitutional boundary.

Kernel v3 admits adapters only through the six-method contract. Every method returns evidence to the worker; only the worker can advance the state.

Conformance suite

The adapter is useful only if it cannot cheat.

The SDK surface is designed for serious integrators: platform teams, evaluators, auditors, model providers, and enterprise workflow owners.

01

Authority firewall

Adapters never decide terminal authority. They submit typed evidence; the state machine decides whether the next transition is even admissible.

02

Evidence-first execution

Every proposal, receipt, judgment, challenge, and review certificate receives a canonical payload commitment before it can enter the Chronicle.

03

Replayable integration

Mission bundles export public role keys, envelope history, state transitions, and commitments so another verifier can replay the same record.

04

Fail-closed onboarding

Unsupported scope, malformed evidence, missing signatures, edited payloads, or forbidden authority claims quarantine the adapter path.

Reference implementation

Production adapters can be powerful without being sovereign.

The contract below is intentionally small. The authority model is not.

What a production adapter adds:authenticated evidence inputs, enterprise policy context, external evaluator artifacts, and independent verification receipts.
enterprise-evaluator-adapter.jsKernel v3 contract
class EnterpriseEvaluatorAdapter extends AdapterContract {
  async initialize(context) { return this.bindPolicy(context); }
  async propose(input) { return this.createEvaluationPlan(input); }
  async execute(input) { return this.runBoundedChecks(input); }
  async evaluate(input) { return this.scoreClaims(input); }
  async produceEvidence(input) { return this.sealArtifacts(input); }
  async verifyEvidence(input) { return this.verifyProvenance(input); }
}

register(new EnterpriseEvaluatorAdapter({
  authority: 'VALIDATION_ONLY',
  externalActions: 0,
  settlementAuthority: 'NONE_GRANTED',
  memoryPromotion: 'HUMAN_ONLY'
}));
