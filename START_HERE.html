GoalOS AGIALPHA Ascension v4.3

Open this first.

This is the institutional audit-candidate package for re-implementing α-AGI Ascension with GoalOS and the existing AGIALPHA token on Ethereum.

Status

  • Audit candidate: yes.
  • Ethereum Sepolia rehearsal path: yes.
  • Uses existing AGIALPHA on mainnet: yes.
  • Mainnet authorized: no.

First commands

npm install
cp .env.example .env
npm run compile
npm test
npm run test:all
npm run static-check

Sepolia

For Sepolia rehearsal, leave AGIALPHA_TOKEN_ADDRESS blank to deploy MockAGIALPHA automatically.

npm run deploy:ethereum-sepolia

Mainnet

Mainnet remains blocked until automated security/toolchain review, internal security review, legal, tax, security, treasury, public-claims, Sepolia evidence, AGIALPHA token verification, and founder approval gates are complete.
