# Mainnet Blockers

Mainnet is **NOT AUTHORIZED** unless all real gates are complete:
- Legal, tax, public-claims, treasury, founder approval hashes.
- External audit closure hash.
- AGIALPHA token verification hash and exact mainnet token address.
- Completed Sepolia rehearsal evidence hash.
- No unresolved critical/high findings and no unaccepted medium findings.
- Explicit founder deploy approval immediately before deployment.
