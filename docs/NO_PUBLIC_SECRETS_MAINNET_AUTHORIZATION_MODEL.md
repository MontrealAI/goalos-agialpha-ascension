# No-Public-Secrets Mainnet Authorization Model

Public GitHub is a public evidence layer. It must not store RPC URLs, private keys, founder signatures, founder/treasury/admin addresses, wallet metadata, private operator notes, or private evidence appendices.

The private operator layer runs locally from gitignored `.private/` inputs and emits only redacted public commitments into `qa/`.

Ethereum Mainnet authorization can become YES only when the public checker validates redacted evidence commitments generated by private operator tooling. The public commitments prove that private evidence exists without publishing the private evidence itself.

No Ethereum Mainnet deployment is performed by GitHub Actions.
