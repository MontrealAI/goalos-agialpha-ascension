# No-Public-Secrets Mainnet Authorization Model

This repository is public. Every committed file, pull request comment, GitHub Actions log, and workflow artifact is treated as public evidence. The authorization model therefore splits Ethereum Mainnet readiness into two layers.

## Public GitHub Evidence Layer

Public GitHub may contain only:

- deterministic local tests and local-chain rehearsal outputs;
- compile, static, security-toolchain, and repository safety reports;
- public-safe hashes, commitments, attestations, pointers, and summaries;
- redacted Evidence Docket commitments under `qa/public-*.json`;
- authorization checkers that can evaluate redacted public commitments.

Public GitHub does **not** need and must not require RPC URLs, private keys, raw founder approval signatures, founder/treasury/admin/deployer/vault/security/community addresses, wallet metadata, private ceremony details, private operator notes, or private evidence appendices.

## Private Operator Layer

Private operators keep sensitive inputs and raw evidence locally under gitignored `.private/` paths. The private layer may contain:

- `.private/mainnet-operator-input.json`
- `.private/mainnet-operator.env`
- `.private/sepolia-rehearsal-private.json`
- `.private/mainnet-preflight-private.json`
- `.private/address-ceremony-private.json`
- `.private/founder-approval-private.json`
- `.private/final-deploy-private.env`

The private tools emit only redacted public commitments into `qa/`. The redacted files contain pass/fail fields, chain target, commit SHA, the fixed Ethereum Mainnet AGIALPHA token address, and hashes of private evidence. They must not contain private keys, RPC URLs, private addresses, signatures unless the founder explicitly chooses publication, raw private artifacts, or private operator notes.

## Decision rule

`TECHNICALLY_MAINNET_READY`, `MAINNET_DEPLOYMENT_AUTHORIZED`, and `ETHEREUM_MAINNET_AUTHORIZED` may become `YES` only when public checkers validate public-safe redacted commitments generated by the private operator tools. Until then the expected public result is `NO` with `PRIVATE_OPERATOR_EVIDENCE_PENDING`.

The deterministic local rehearsal runs on the non-forked Hardhat local chain, deploys MockAGIALPHA plus the proof-work registries, completes the proof-work loop and a negative path, and writes only local-simulation manifests with address commitments rather than raw operator addresses.

GitHub Actions never deploys Ethereum Mainnet and never mints or deploys a new AGIALPHA token. Ethereum Mainnet AGIALPHA remains `0xA61a3B3a130a9c20768EEBF97E21515A6046a1fA`.


## v4.4 implementation note

`hardhat.config.ts` never throws during public config load for missing private RPC URLs or private keys. Private values are required only by explicit preflight/deployment scripts. Public GitHub may contain schemas, templates, reports, and hashes; `.private/` contains filled operator inputs and remains ignored.
