# Public Governance Deployment Approval

Public governance approval is represented by `qa/public-governance-approval-evidence.json`. No private founder signature is required for public YES. Approval requires merged/committed public governance evidence, required checks, branch protection or public risk acceptance, sensitive-path policy or risk acceptance, generated status docs, and release plan `mainnet-authorization-vX.Y.Z`.

Release notes must state: not externally audited, not deployed, authorized for manual gated Ethereum Mainnet deployment, deployment still requires runtime RPC/key, no new AGIALPHA token is deployed, and AGIALPHA token address is fixed.
