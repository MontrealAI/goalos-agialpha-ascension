# Governed Decision State

A **Governed Decision State** is the core Mission OS deliverable.

A report says what an AI found. A governed decision state records what was claimed, what supports it, what contradicts it, what risks remain, what action follows, what proof is attached, what can be reused, and what must happen next.

## Minimum contents

- Mission Contract
- Claims Matrix
- Source Provenance
- Contradiction Register
- Evidence Docket
- Verifier Report
- Risk Ledger
- Executive Brief
- Decision Deck
- Action Graph
- Chronicle Entry
- Capability Package

## Rule

> The deliverable is not a document. The deliverable is a governed decision state.
