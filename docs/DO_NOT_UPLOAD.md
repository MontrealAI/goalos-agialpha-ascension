# Do Not Upload

Do not upload any of the following to GitHub:

```text
paid buyer products
GoalOS product ZIPs sold to customers
private keys
seed phrases
.env
RPC secrets
API keys
wallet backups
customer data
private Evidence Dockets
legal memos
tax memos
unredacted audit findings
private commercialization agreements
```

The repository is for implementation, documentation, schemas, tests, and public-safe evidence scaffolding only.
