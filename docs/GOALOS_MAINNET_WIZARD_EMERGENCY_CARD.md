# GoalOS Mainnet Wizard Emergency Card

Never rerun deployment blindly.
Never share your private key or seed phrase.
Use --status, --resume, --verify or --recover.

Useful commands:

```bash
npm run goalos:mainnet:wizard -- --status
npm run goalos:mainnet:wizard -- --resume
npm run goalos:mainnet:wizard -- --verify
npm run goalos:mainnet:wizard -- --recover
```

If verification is pending, do not redeploy. Resume verification.

If interrupted, do not restart from the beginning. Resume safely.

Production is not activated by initial deployment. Do not accept user funds.
