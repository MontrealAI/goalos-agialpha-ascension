# Sepolia Rehearsal Evidence Docket v4.2

This docket is the required evidence package for moving from audit candidate to mainnet discussion.

## Required evidence

1. Compile output
2. Full test output
3. Static QA output
4. Deployment transaction hashes
5. Deployed contract addresses
6. AGIALPHA token mode used on Sepolia
7. Proof mission posted transaction
8. Builder claim transaction
9. Proof submission transaction
10. Reviewer bond transaction
11. Review approval transaction
12. Proof Card registration transaction
13. Credential issuance transaction
14. Reputation update transaction
15. Reward / bond settlement transaction
16. Evidence Docket hash
17. Public claims boundary note
18. Operator notes

## Proof loop

```text
Sponsor posts proof mission
Builder claims mission
Builder submits proof
Reviewer validates proof
Proof Card registers
Credential issues
Reputation updates
Rewards/bonds settle
Evidence Docket publishes
```

## Completion rule

No mainnet discussion without a complete Evidence Docket.
