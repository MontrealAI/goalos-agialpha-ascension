# Public Claims Review v4.2

```text
uses existing AGIALPHA token
runs private intelligence off-chain
anchors public-safe proof on-chain
coordinates proof missions
supports proof cards, credentials and reputation
is an audit candidate
mainnet is gated
```

## Prohibited public claims

```text
audited
mainnet ready
guaranteed non-security
guaranteed value
investment opportunity
yield
revenue share
autonomous superintelligence achieved
Kardashev Type II achieved
```

## Required boundary statement

```text
The intelligence stays private. The proof becomes verifiable. This package is not audited and is not mainnet authorized.
```
