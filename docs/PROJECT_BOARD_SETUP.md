# GitHub Project Board Setup

Create a project board named:

```text
GoalOS AGIALPHA Ascension Production Path
```

## Columns

```text
Inbox
Ready
In Progress
Blocked
Review
Evidence Required
Done
Archived
```

## Fields

```text
Gate
Owner
Evidence Link
Risk
Status
Target Date
Reviewer
Mainnet Impact
```

Any item with `Mainnet Impact = yes` remains blocked unless gate evidence exists.
