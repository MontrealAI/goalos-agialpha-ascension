# Founder / Non-Technical Operator Guide v3.0

## What this package does

It lets engineers deploy GoalOS AGI Ascension contracts on Ethereum using existing AGIALPHA.

## What it does not do

- It does not deploy AGIALPHA.
- It does not mint AGIALPHA.
- It does not automatically fund vaults.
- It does not authorize mainnet.
- It does not put private prompts or customer data on-chain.

## Ask the engineer for

1. Compile output.
2. Test output.
3. Static QA output.
4. Sepolia deployment manifest.
5. Sepolia proof-of-workflow evidence docket.
6. AGIALPHA token verification output.
7. External audit closure.
8. Legal/tax/public-claims signoffs.
9. Treasury/founder/admin address ceremony evidence.
10. Final founder approval memo.

## Simple go/no-go rule

If any gate is missing, mainnet is not authorized.
