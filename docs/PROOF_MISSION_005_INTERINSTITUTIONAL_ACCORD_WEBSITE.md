# Public Proof Mission 005 website integration

This additive overlay publishes **The Interinstitutional Accord** after the canonical v86 site and Public Proof Missions 001–004 are generated.

It adds:

- `proof-mission-005.html`;
- a five-mission `proof-missions.html` portal;
- one marked Mission 005 homepage panel;
- a promoted Mission 005 block on Mission 004;
- public, non-evidence Mission 005 templates and a 40-stage Mainnet route CSV;
- deterministic verification, regression tests, and desktop/mobile browser QA.

It does not edit Solidity contracts, package files, deployment scripts, Mainnet deployment records, or `website/v86_actual_site/**`.

## Claim boundary

Mission 005 is a protocol publication. It does not claim completed federation, external audit, legal approval, achieved AGI/ASI, or any empirical mission result. Every generated template is marked `TEMPLATE_NOT_EVIDENCE`.
