# Upload Batch Guide

GitHub browser uploads are limited, so this pack is split into four batches.

## Batch 01 - root + GitHub metadata

```text
BATCH_01_ROOT_GITHUB_METADATA__SELECT_CONTENTS_AND_UPLOAD
```

Commit message:

```text
Initialize root files and GitHub governance metadata
```

## Batch 02 - docs

```text
BATCH_02_DOCS__SELECT_CONTENTS_AND_UPLOAD
```

Commit message:

```text
Add GoalOS AGIALPHA Ascension documentation
```

## Batch 03 - contracts, schemas, audit, QA

```text
BATCH_03_CONTRACTS_SCHEMAS_AUDIT__SELECT_CONTENTS_AND_UPLOAD
```

Commit message:

```text
Add contracts, schemas, audit handoff, and QA artifacts
```

## Batch 04 - scripts, tests, app, deployments

```text
BATCH_04_SCRIPTS_TESTS_APP__SELECT_CONTENTS_AND_UPLOAD
```

Commit message:

```text
Add scripts, tests, app shell, and deployment helpers
```

## Upload rule

Select the contents of the batch folder, not the batch folder itself.
