# Initial Mainnet Limited Release

Authorization profile: `INITIAL_MAINNET_LIMITED_RELEASE`.

Purpose: deploy, verify, establish Ledger ownership, and gather real Mainnet operational evidence.

This profile does **not** authorize production use, user funds, customer onboarding, protocol activation, Phase-B configuration, settlement, a public frontend, production announcements, public reliance, or unbounded economic exposure. Production activation requires `CONTROLLED_PRODUCTION_RELEASE` with a new authorized certificate, new configuration root, and Ledger-signed activation/configuration transactions.

Not externally audited.

Source of truth: `config/authorization-profiles/initial-mainnet-limited-release.json` and `qa/mainnet-authorization-certificate.json`.
