# Non-Technical Operator Runbook v2.0

## What this package does

It lets engineers deploy GoalOS AGIALPHA Ascension contracts that use the existing AGIALPHA token.

## What it does not do

It does not create a new AGIALPHA token.
It does not automatically fund vaults.
It does not authorize mainnet deployment.
It does not replace legal/tax/audit review.

## Ask your engineer for

- compile output
- test output
- static check output
- Sepolia deployment manifest
- Sepolia proof-work loop evidence
- AGIALPHA token verification output
- automated security/toolchain clearance
- internal security review
- mainnet preflight output
