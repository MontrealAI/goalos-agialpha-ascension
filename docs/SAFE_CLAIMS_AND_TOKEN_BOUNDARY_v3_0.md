# Safe Claims and Token Boundary v3.0

## Approved public line

GoalOS decides what may evolve. AGIALPHA coordinates proof-settled work. Evidence Dockets make claims auditable.

## Approved descriptions

- GoalOS improves workflows.
- AGIALPHA coordinates proof work.
- Proof Cards create trust.
- Credentials create reputation.
- Reputation unlocks better jobs.
- Evidence Dockets separate public proof from private intelligence.
- Private prompts, traces, documents, and customer data stay off-chain.

## Prohibited claims

- investment return
- profit
- yield
- revenue share
- passive income
- price target
- guaranteed resale value
- guaranteed non-security
- achieved AGI / ASI
- autonomous sovereignty achieved
- audited unless audit is complete
- legal/tax approved unless memos exist
- mainnet authorized unless gates are complete
