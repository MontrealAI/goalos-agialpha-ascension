# Best Initial Use Cases

## 1. Customer Support Proof Room

Fastest path to Proof Card 001.

```text
support cases → workflow v1.0 → score → diagnosis → workflow v1.1 → review → Proof Card
```

## 2. Claims-Safe Launch Content

Prevents unsafe public language before launch.

```text
draft copy → claims scan → rewrite → reviewer approval → public-safe copy
```

## 3. Repository + Website Documentation QA

Turns the repository into an auditable public trust surface.

```text
README/docs/site → QA → fixes → validation → Proof Card
```

## 4. Commercialization Proof-of-Value

Makes partner compensation evidence-based.

```text
lead → sponsor → proof job → proof card → report → tranche decision
```

## 5. Builder / Reviewer Credentialing

Creates the first credible labor network.

```text
starter proof job → review → credential → reputation → better jobs
```
