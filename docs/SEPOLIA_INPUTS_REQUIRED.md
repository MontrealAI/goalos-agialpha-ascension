# Sepolia Inputs Policy (Not Committed)

Status: **OPTIONAL_RUNTIME_REHEARSAL_INPUTS_ONLY**.

Ethereum Sepolia replay inputs are optional deployer inputs. They must stay in `.private/` or the operator's local environment and must not be committed, added to public workflow logs, or uploaded as public artifacts.

Public Sepolia deployment is recommended but not mandatory for public repository authorization. Local deterministic rehearsal, public-safe Evidence Docket artifacts, toolchain clearance, mainnet-shaped simulation, and the Mainnet Authorization Certificate are sufficient for the public YES state.
