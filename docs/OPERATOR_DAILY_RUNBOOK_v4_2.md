```text

```

## Incident response

1. Pause affected module if necessary.
2. Capture transaction hash and block number.
3. Capture role / signer involved.
4. Capture public-safe incident summary.
5. Capture private evidence appendix.
6. Record mitigation.
7. Record rollback / recovery action.
8. Update Evidence Docket.
