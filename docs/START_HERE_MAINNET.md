# Start Here: Mainnet

Run `npm run mainnet:status`, then `npm run mainnet:local-checks`, `npm run mainnet:security`, `npm run mainnet:local-rehearsal`, `npm run verify:agialpha-token:public`, and `npm run mainnet:public-authorize`. Final deployment, if authorized, is `npm run deploy:ethereum-mainnet:gated` from a local private environment only.
