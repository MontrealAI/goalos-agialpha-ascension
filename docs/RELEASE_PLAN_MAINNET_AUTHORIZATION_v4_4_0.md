# Release Plan — Mainnet Authorization v4.4.0

Release tag: `mainnet-authorization-v4.4.0`

Release title: **GoalOS AGIALPHA Ascension v4.4.0 — Certificate-Backed Manual Ethereum Mainnet Authorization**

This release packages certificate-backed public authorization for manual, local, gated Ethereum Mainnet deployment. Runtime RPC/key material is required only at local broadcast time and is not stored in GitHub. CI cannot deploy mainnet.

The canonical AGIALPHA token address is `0xA61a3B3a130a9c20768EEBF97E21515A6046a1fA`. No new AGIALPHA token is deployed.

Public Sepolia is recommended but not mandatory for public authorization. Branch protection is public-risk accepted, with pre-deployment hardening recommended. Research paper v6.3 is published under `docs/papers/sovereign-rsi/v6.3/`.
